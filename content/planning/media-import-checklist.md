# Media Import Checklist

- Verify asset came from approved `media-production-plan.v1.json`.
- Verify no real client data or personal data appears in the asset.
- Verify dimensions match the required export size.
- Verify alt text exists in English and Arabic.
- Upload asset to Sanity only from `tanzkit_website_v2`.
- Map the returned `sanityAssetRef` into `homepage-media.v1.json` or `solutions-media.v1.json`.
- Run a dry-run preview before seeding related draft content.
- Do not publish.
